These are some of the open source projects that I've built over the past few years:

  • Wombat: Lightweight Ruby web crawler/scraper with an elegant DSL which extracts structured data from pages.
  • AsymmetricGridView: Android ListView that mimics a GridView with asymmetric items
  • EliFUT: A "football manager" style Android game
  • ktnes: A multiplatform NES emulator written in Kotlin
  • GifImageView: Android ImageView that handles animated GIF images
  • Random Maze Generator: Find the shortest path out of a generated random maze.
